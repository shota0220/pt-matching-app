# pt-matching-app

理学療法士（PT）と利用者を、具体的な症状に基づいたマッチングロジックでつなぐものです。

## プロジェクトの概要

本システムは、利用者の悩み（膝の痛み、脳卒中後遺症など）に対し、最適な専門性を持つ理学療法士を優先的に表示し、スムーズなマッチングを実現するために開発しました。

50名分のサンプルデータを搭載しており、実際の運用に近い形での検索シミュレーションが可能です。

## ローカル環境でのセットアップ手順

プロジェクトのルートディレクトリ（`package.json` がある場所）で順に実行してください。

ターミナルに以下をコピー＆ペーストするだけで準備が完了します。

```bash
# 依存ライブラリのインストール
npm install

# 開発サーバーの起動
npm run dev
