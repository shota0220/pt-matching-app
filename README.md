# pt-matching-app
これは、利用者の具体的な悩み（膝の痛み、脳卒中後遺症など）に対し、最適な専門性を持つ理学療法士を優先的に表示してマッチングするものです。

https://github.com/shota0220/pt-matching-ap

gitコマンドが使える場合は以下を実行してください。
  https://github.com/shota0220/pt-matching-app.git

プロジェクトのフォルダ内でターミナルを開き、以下を実行してください。 
 npm run dev

ブラウザで以下のURLを開いてください
http://localhost:3000
